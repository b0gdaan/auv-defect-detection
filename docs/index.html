System Architecture
AUV ↔ Shore Station
🎥
Step 1 · AUV

Dual Camera Capture

Two SJ4000 Action Cameras via V4L2. Front (inspection) + Nadir (downward). Raspberry Pi 4 reads both simultaneously.

🗜️
Step 2 · AUV

Huffman Compression

JPEG frames further compressed with Huffman coding, reducing payload to 61–74% of JPEG size before wireless transmission.

📡
Step 3 · WiFi

TCP Streaming

TP-Link TL-703N access point at 192.168.0.1. AUV transmits on port 9000 to shore station. Range: ~80m at surface.

🧠
Step 4 · Shore

CNN Inference

MobileNetV2 backbone classifies each frame into 4 defect classes at ~115ms/frame. Annotated feed displayed live.

📝
Step 5 · Shore

Defect Logging

Detections logged to CSV with timestamp, class, confidence. Bounding box overlay drawn with OpenCV in real time.

Neural Network
MobileNetV2 Architecture

Pretrained on ImageNet, fine-tuned on ~1,200 labelled AUV frames with underwater augmentation.
